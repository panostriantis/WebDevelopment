README


Project Α.Α : 2015-2016


Στο αρχείο e-r.pdf είναι ένα σχεσιακό διάγραμμα που περιγράφει τη δομη της βάση μας.

Με το createANDentries.php δημιουργείται η βάση με τα entries.

Τα αρχεία για τη δημιουργία της βάσης είναι με τη σειρά

1) Δημιουργία βάσης teliko_db.php

2) Δημιουργία πινάκων (Σταθμός και Ρύπος) teliko_tables_stathmos_rupos.php

3) Δημιουργία πίνακα (χρήστες) teliko_table_users.php

4) Εισαγωγή Τιμών στους πίνακες (Σταθμός και Ρύπος) και (Χρήστες) με τα αρχεία
insert_stathmos_rupos.php & insert_users.php αντίστοιχα.

5) Με το αρχείο teliko_tables_arxeiwn_csv.php εισάγουμε όλα αρχεία του υπουργείου στη βάση μας.

6) Στο αρχείο ypeka_api.php δημιουργούμε το API όπως ζητείται.

7) Στο αρχείο systemServer.php δημιουργούμε ένα σύστημα που διαχειρίζεται τους χρήστες και
τις υπηρεσίες που χρειάζονται (admin & users) όπως : login,signup,statistics,upload,
deleteStation,addStation.

8) Στο αρχείο accessSite.html έχουμε μία σελίδα στην οποία ένας χρήστης μπορεί να
εγγραφεί στο σύστημα ή να συνδεθεί. Οι admin είναι static και κάνουν από την ίδια  σελίδα σύνδεση.

9) Στο αρχείο systemSite.html έχει πρόσβαση κάποιος χρήστης ή διαχειριστής που έχει συνδεθεί.
Στη συγκεκριμένη σελίδα μπορεί να χρησιμοποιήσει κάποιες υπηρεσίες που παρέχονται.

10) Στο αρχείο demoWebsite.html είναι το demo Site που ζητείται στην άσκηση.

11) Στο αρχείο teliko_db_mikro.sql είναι ένα ενδεικτικό export της βάσης μας γιατί το κανονικό
export είναι πολύ μεγάλο.

Τελος να προσθέσω πως τα αρχεία ειναι αυτούσια όπως έχουν αναπτυχθεί στο σύστημα μας 
(δηλαδή με paths του συστήματος μας π.χ C:\\wamp\\www\\panos\\mydatabase\\BenzIOA2010.dat)
όπου ο φάκελος mydatabase περιέχει όλα τα .dat αρχεία του υπουργείου.
